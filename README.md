# TrustList

Verified real estate marketplace prototype (frontend only) — BSIT capstone
project, University of Nueva Caceres. See [CLAUDE.md](./CLAUDE.md) for the
full project spec, design direction, and build order.

## Stack

React + TypeScript + Vite + React Router. No real backend — `src/mocks/`
holds hardcoded data and `src/services/` exposes it through thin async
functions that simulate network delay.

## Getting started

```bash
npm install
npm run dev
```
